问题

如何在hexo插入图片

解决

修改配置 _config.yml

post_asset_folder: true

安装插件

npm install hexo-asset-image –save

修改插件

找到如下文件，先备份

/node_modules/hexo-asset-image/index.js

然后改为以下内容，如果不修改，图片显示不正确

测试

hexo new post hexo中插入图片
